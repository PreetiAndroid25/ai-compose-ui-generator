# AI-Driven UI Automation for Android (Jetpack Compose)

This project demonstrates how **AI can automate Android UI development** using
**natural language** and **Jetpack Compose**.

By providing plain English screen descriptions, AI generates
**production-ready Compose UI**, **ViewModels**, and **UiState** following
**MVVM architecture**, accessibility, and theming best practices.
