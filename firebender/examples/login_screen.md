# Login Screen

Create a login screen with the following requirements:

UI Elements:
- Email input field
- Password input field
- Login button

Behavior:
- Disable login button when fields are empty
- Show loading indicator when login is in progress
- Show error message if login fails

State:
- Idle
- Loading
- Error with message
- Success

Accessibility:
- Content descriptions for inputs and button
- Keyboard actions for email and password fields

Theme:
- Material 3
- Supports light and dark mode
